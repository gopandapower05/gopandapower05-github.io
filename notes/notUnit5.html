Lesson 2.1


Strings

if the string will include double quote use backslash (\)
for example:

double quote: \""
single quote: \'
backslash: \\
tab: \t
carriage reurn: \r
can add the strings to create new string value (is called concatenation)
you can use the lowercased() method to normalize the print
we could also use count property where it counts the number or letters
