Lesson 1.3


Operator

Operator is used for modify or reassign a value.
We can use + - * / for basic functions.
we can use = to perform arithmetic.
for decimals, use Double.
Integers uses whole number so the answer will round to the nearest whole number.
We can use a compound assignment operator, which adds the = operator after an arithmetic operator.
use brackets ex) (3+4)*8.
If want to convert integers to double use brackets
ex) Double(x) + y
