Lesson 1.2

Constants and associate a name with a value of a particular type. Can then use the name to refer to that value many times in program. Once it's set, the value of a constant cannot be changed. But the value of a variable can be changed at any time. Asking it to associate the constant or variable name with the assigned value so the programmer can access the value later.


Constant

Create constant a constant using let name = "whatever" Name is constant so you can not give a new value after this


Variables

Use variable when the program has a value that may change during the life time. Create a new variable by using var age = "whatever"


Constant and Variable

- names can not contain mathematical symbols
- names can not contain space
- names can not begin with a number

Types

there are 4 types Integer = represents whole numbers or integers
Double = Represents numbers requiring decimal points
Booleon = Represents true of false
String = Represents text
Can not mix them up or add them together. IF they are written together, the console will not recognize and will give error And always give value for string, double, integer, and booleon.
