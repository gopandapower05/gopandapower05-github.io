Lesson 1.4


Control Flow

Each if statement uses a logical operator to decide if something is true or false
for example:

We can use == when two items must be equal
we can use != when values must not be equal to each other
we can use > when value on the left must be greater than the value on the right
we can use >= when value on left must be greater than or equal to the value on the right
we can use < when value on the left must be less than the value on the right
we can use <= when value on the left must be less than or equal to the value on the right
we can use && when the conditional statement on the left and right must be true
we can use || when the conditional statement on the left or right must be true
if statement = if this condition is true, then run this block of code.
