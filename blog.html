Lesson 3: Images

img= image body element, defines an image in a HTML page.
src= source attribute, specifies the URL of an image. There are 2 different ways to use this attribute.

  1. using URL of image: Method #1
  2. Saving the Image File: Method #2

advantage of using the second method is if the image is deleted in the internet, the image won't be displayed.
Height&Width: Used to resize the image in pixels. You can specify only one height or width and the aspect ratio will be adjusted.
